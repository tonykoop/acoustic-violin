# Open String Bow Exercise

**Provenance:** Original exercise.

**Purpose:** Before fretting a single note, spend 5 minutes on open strings.
This is the most important exercise for a new bow arm.

**How to play:**
- Use the full bow on each half note: start at the frog, draw smoothly to the tip.
- Listen for a clear, centred tone — not scratchy, not airy.
- Sequence: G3 → D4 → A4 → E5 → E5 → A4 → D4 → G3 (up then back down).

**Practice tips:**
- Keep the bow parallel to the bridge at all times (watch it in a mirror or
  phone camera).
- Right elbow height: rises as you move from G string to E string.
- "Slow bow, fast rosin" — more bow speed gives brightness; more weight gives
  depth. Experiment on open strings before adding left-hand complexity.
