# Acoustic Violin — Learn to Play

Starter songbook for Tony Koop's acoustic violin build.

## Tuning

Standard violin tuning — **GDAE** (low to high):

| String | Pitch | Colour convention |
|--------|-------|-------------------|
| 4 (lowest, leftmost) | G3 | |
| 3 | D4 | |
| 2 | A4 | |
| 1 (highest, rightmost) | E5 | |

Tune with a chromatic tuner clipped to the scroll. Tune G first (lowest),
then tune each string upward in perfect fifths.

## First-position range

| String | Open | 1st finger | 2nd finger | 3rd finger | 4th finger |
|--------|------|-----------|-----------|-----------|-----------|
| G | G3 | A3 | B3 | C4 | D4 |
| D | D4 | E4 | F♯4 | G4 | A4 |
| A | A4 | B4 | C5 | C♯5 | D5 |
| E | E5 | F♯5 | G5 | G♯5 | A5 |

All tunes in this songbook stay in **first position**.

## Bowing notation

- `↓` = down-bow (frog toward tip)
- `↑` = up-bow (tip toward frog)
- Default: **down-bow on every downbeat**; up-bow on upbeats.
